## 项目简介（GitHub 描述，100字内）

**中文（100字内）：**
FluxSQL 是一款 AI 驱动的数据库建表工具，支持 Oracle、MySQL、PostgreSQL、SQL Server 四大主流数据库。只需用自然语言描述表结构，AI 自动生成完整的字段定义和 SQL 语句。纯前端实现，下载即用，数据完全本地存储。

**English (under 100 words):**
FluxSQL is an AI-powered database table designer. Describe your tables in plain language and get clean, dialect-aware SQL instantly. Supports Oracle, MySQL, PostgreSQL, and SQL Server. No installation, no server — just open `index.html` in your browser and start building. Your data stays local, always.

## 核心卖点 / Key Selling Points

### 中文
1. 🤖 **AI 智能建表** — 自然语言描述，自动生成完整表结构，告别手写 DDL
2. 🗄️ **四大数据库** — 一键切换 Oracle / MySQL / PostgreSQL / SQL Server 方言
3. ⚡ **即开即用** — 纯 HTML 单文件，下载后直接在浏览器打开，零配置零安装
4. 🔒 **完全本地** — 数据存储在浏览器本地，无服务器，无数据上传，隐私安全
5. 🎨 **可视化编辑** — 直观的表格编辑界面，支持拖拽排序字段
6. 📤 **灵活导出** — 支持导出 SQL（DDL）和 JSON 格式，导入现有 SQL 结构
7. 💡 **智能提示** — 内置快捷描述模板，快速生成常见表结构（用户、订单、商品等）

### English
1. 🤖 **AI Table Generation** — Describe in natural language, get complete DDL instantly
2. 🗄️ **Multi-Database Support** — Switch between Oracle / MySQL / PostgreSQL / SQL Server
3. ⚡ **Zero Setup** — Single HTML file, open in browser, works immediately
4. 🔒 **100% Local** — No server, no uploads, data stays in your browser's localStorage
5. 🎨 **Visual Editor** — Intuitive table editor with drag-and-drop field ordering
6. 📤 **Export Anywhere** — Export as SQL (DDL) or JSON, import existing SQL schemas
7. 💡 **Smart Templates** — Built-in prompt chips for common tables (users, orders, products)

## 使用场景 / Use Cases

### 中文
- **数据库设计阶段**：快速原型设计，把需求文档转化为数据库结构
- **多数据库项目**：一次设计，快速切换不同数据库的 DDL 语法
- **数据库学习**：直观了解不同数据库的字段类型和语法差异
- **团队协作沟通**：用可视化界面向非技术人员展示数据库设计
- **SQL 面试备战**：快速生成练习用的表结构和测试数据 schema
- **遗留系统整理**：导入现有 SQL，在可视化界面中整理和优化表结构

### English
- **Database Design** — Rapidly prototype schemas from requirements
- **Multi-DB Projects** — Design once, export to any SQL dialect
- **Learning SQL** — Visualize field types and syntax differences across databases
- **Team Communication** — Present DB design visually to non-technical stakeholders
- **Interview Prep** — Quickly generate practice schemas and table structures
- **Legacy Refactoring** — Import existing SQL, visualize and optimize
